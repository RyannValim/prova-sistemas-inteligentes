## Para rodar

##### Criar e ativar o ambiente .venv

**Criação**: python -m venv .venv

**Ativação Linux**: source ./.venv/bin/activate
**Ativação Windows**: ./.venv/Scripts/activate

## Instalação de dependências

**Na raiz do projeto**: pip install -r requirements.txt

## Para rodar:

**Na raiz do projeto**: python main.py
